## Description
Brain Games is a collection of console mini-games to train your brain. Each game asks questions that need to be answered correctly. After three correct answers, the game is considered completed. Incorrect answers end the game and prompt you to play it again.

## Installation

```bash
npm install
npm link
```

## Games

### Brain Even
Determine if the number is even or not.

**How to play:**
```bash
brain-even
```
